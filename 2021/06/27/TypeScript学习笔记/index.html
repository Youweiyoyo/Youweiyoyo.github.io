TypeScript学习笔记

发表于 2021-06-27 更新于 2021-07-04 分类于 TypeScript

1、TypeSCript 简介

TypeScript 是 javaSCript 的超集，它对 JS 进行了扩展，往 JS 中引入了类型的概念，并添加了许多新的特性，对比 JS 而言，TS拥有静态类型，更加严格的语法、更强大的功能

2、基本类型

  1. 类型声明

    • 类型声明是 TS 非常重要的一个特点

    • 通过类型声明可以指定 TS 中变量(参数、形参)的数据类型

    • 指定类型后，为变量赋值时，TS 编译器会制动检查值是否符合类型声明，符合则赋值，否则会在编辑器上出现波浪线提示错误

  2. 类型

    1. Number 类型

    2. String 类型

    3. Boolean 类型

    4. 字面量类型，直接使用字面量进行赋值

    5. any 类型，任意类型。设置类型为 any 后相当于关闭了 Ts 的类型检查功能

    6. unknown 类型，未知类型的值

      • unknown 实际上就是一个类型安全的 any
      • unknown 类型的变量，不能直接赋值给其他变量

    7. void 类型，用来表示空值，以函数为例，表示没有返回值

    8. never 类型，表示永远不会返回结果

    9. object 类型

    10. array 类型

    11. tuple 元组，固定长度的数组

    12. enum 枚举

    1. | 链接多个类型，满足一个类型即可

    2. & 链接多个类型，必须同时满足

    给类型起别名，简化类型的使用

3、创建类

类可以理解为对象的模型，程序中可以根据类创建指定类型的对象

  1. 静态属性：
    • 通过 static 可以声明一个静态属性，或者又称为类属性
    • 该属性通过类直接进行调用

4、类的继承

子类可以通过继承来使用符类中的属性

  1. 通过 extends 关键字可以继承符类的属性

  2. 如果子类的方法名称和父类的方法名称一样，叫做重写，优先使用自己的方法

  3. 如果在子类中使用了 constructor 构造函数，必须使用super() 关键字对父类中的方法进行调用

5、封装类

对象实质上就是属性和方法的容器，它的主要作用就是存储属性和方法，这就是所谓的封装

  1. 只读属性：readonly 修饰符：如果在声明属性时添加一个readonly，则属性便成了只读属性无法修改

  2. 公共属性：public 修饰符（默认）：规定属性可以在父类、子类中都可以访问

  3. 私有属性：private 修饰符：规定属性为私有属性，只能在当前类进行使用

  4. 保护的属性：protected 受保护的属性，只能在当前类，和当前类的子类中使用
